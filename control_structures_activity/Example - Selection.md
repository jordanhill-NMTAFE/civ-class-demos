Example 1 - Simple if statement:

```python

if age >= 18:
    print("You are eligible to vote.")
```

Example 2 - `if-else` statement:

```python

if rain:
    print("Take an umbrella!")
else:
    print("Enjoy the weather!")
```

Example 3 - `if-elif-else` statement:

```python

if marks < 50:
    print("Fail")
elif 50 <= marks < 60:
    print("Grade D")
elif 60 <= marks < 70:
    print("Grade C")
elif 70 <= marks < 80:
    print("Grade B")
else:
    print("Grade A")
```
