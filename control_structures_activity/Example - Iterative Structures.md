Example 1 - Using a `for` loop:

```python

for number in range(1, 6):
    print(number)
```

Example 2 - Using a `while` loop:

```python
number = 1

while number <= 5:
    print(number)
    number += 1
```

Example 3 - Nested loops:

```python
for row in range(1, 4):
    for column in range(1, 4):
        print(row, column)
```
